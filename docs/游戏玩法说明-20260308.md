
## 一、核心游戏循环设计

```
每日挑战 → 主题关卡推进 → 收集/解锁 → 成就展示 → 社交分享 → 每日挑战
```

---

## 二、主题体系与关卡架构

### 2.1 四大主题方向

| 主题 | 视觉风格 | 内容来源 | 目标用户画像 |
|------|---------|---------|-------------|
| **年代感** | 复古胶片、老海报、怀旧色调 | 80-90年代经典电影海报、老广告、复古插画 | 25-40岁怀旧群体 |
| **漫画连载** | 手绘/插画、奇幻、叙事感 | 古籍卷轴、符文、书房藏经阁等漫画场景 | 喜欢漫画与奇幻的玩家 |
| **国风动画** | 水墨/工笔、东方美学 | 《哪吒》《姜子牙》《长安三万里》等国产动画 | Z世代国潮爱好者 |
| **风景大片** | 自然风光、航拍视角、极简构图 | 国家地理级风光摄影、城市天际线 | 视觉享受型玩家 |

### 2.2 关卡结构设计（每个主题60关）

```
主题解锁线：
├── 新手村（1-10关）：3×3 网格，教学引导
├── 进阶区（11-25关）：4×4 网格，引入道具
├── 挑战区（26-40关）：5×5 网格，复杂图案
└── 大师区（41-50关）：6×6 网格，限时挑战
└── 王者区（51-60关）：7×7 网格，限时挑战


特殊关卡：
- 每10关：BOSS关（动态拼图、异形切割）
- 每主题第50关：全景长图（横向滚动拼图）
```

### 2.3 关卡解锁机制

- **线性解锁**：完成前一关解锁下一关
- **星级评定**：每关1-3星（基于步数效率、时间、是否使用提示）
- **星数兑换**：累积星星可提前解锁后续主题（肝或技巧）
- **每日免费解锁**：观看激励视频可试玩锁定关卡一次

## 三、道具系统设计

### 3.1 基础道具（消耗品）

| 道具 | 功能 | 获取方式 | 使用策略 |
|------|------|---------|---------|
| **放大镜** | 高亮显示一个正确位置的图块（3秒） | 每日签到、关卡奖励 | 开局找锚点 |
| **磁铁** | 自动吸附一个图块到正确位置 | 商店购买、成就奖励 | 最后几块救急 |
| **时间沙漏** | 挑战模式增加30秒 | 看广告获取 | 
| **图块序号角标** | 显示图块序号 | 看广告获取 | 

### 3.3 道具经济循环

```
产出：
├── 关卡首通奖励（大量）
├── 每日任务（稳定）
├── 成就系统（一次性）
├── 观看广告（可选）

消耗：
├── 高难度关卡刚需
├── 追求完美三星
├── 挑战模式冲榜
└── 限时活动竞争
```

## 四、收集与成长体系

### 4.1 图鉴收集

- 每完成一关，该图收入"回忆相册"
- 三星完成解锁高清原图下载/分享
- 集齐一个主题全部50关解锁隐藏彩蛋（动态壁纸/开发者评论）

### 4.2 玩家等级

```
等级经验来源：
- 完成关卡（基础）
- 获得星星（加成）
- 首次通关（额外）
- 每日挑战排名（大量）

等级特权：
- 解锁新主题
- 增加道具携带上限
- 专属头像框
- 高级排行榜显示
```

### 4.3 个性化装扮

- **拼图底板**：不同纹理（木纹、大理石、科技网格）
- **图块样式**：圆角、异形、发光边缘
- **UI主题**：跟随选择的拼图主题自动切换，或手动设置
